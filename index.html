About

Wrench is one of the main character from the game Watch Dogs 2 and I decided it would be a fun project to create a real version of his light up mask.

After some searching, I found a good tutorial on Instructables. However, I did not have access to a 3D printer, wanted the mask to be portable and felt there room for improvement with the code. I therefore designed and coded my own version using the tutorial as inspiration.

This new version has 48 expressions controlled by an Android application.


Parts

What you'll need:

Plastic Wrench mask
Black spray paint
Metallic silver paint
Fine paintbrushes
Arduino UNO
USB A to USB B cable
x2 white 8x8 LED matrix with module
HC-06 wireless bluetooth module
9V battery holder with connector lead
x14 male to female jumper wire, ideally 5 should be black
Arduino UNO plastic case
Velcro strip
Black electric tape
Liquid nails
